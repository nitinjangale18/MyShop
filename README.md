# Advanced Java Web Application

## Technologies Used
- Java (JSP, Servlets)
- JDBC
- MySQL
- Apache Tomcat
- HTML, CSS, JavaScript

## Features
- User Authentication
- Admin Dashboard
- Product Management
- Cart & Orders

## How to Run
1. Import project into Eclipse
2. Configure Tomcat Server
3. Setup MySQL database
4. Run on server

## Note
WAR file is generated locally for deployment.
